# CloudSync Pro - Security & Privacy

At CloudSync Pro, security and privacy are our top priorities. This document outlines our security measures and privacy practices.

## Data Encryption

### In Transit
- All data transfers use TLS 1.3 encryption
- Perfect Forward Secrecy (PFS) enabled
- Certificate pinning in mobile apps

### At Rest
- AES-256 encryption for all stored files
- Encryption keys are managed per-customer
- Keys are rotated every 90 days
- Zero-knowledge architecture available for Enterprise

## Access Control

### User Authentication
- Password requirements: minimum 8 characters, including uppercase, lowercase, number, and symbol
- Password hashing with bcrypt (cost factor 12)
- Session timeout after 30 minutes of inactivity (configurable)
- IP-based access restrictions (Enterprise)

### Two-Factor Authentication (2FA)
- TOTP-based authenticator app support (Google Authenticator, Authy, etc.)
- SMS backup codes (not recommended for high-security accounts)
- Hardware security key support (YubiKey, Google Titan)
- Mandatory 2FA enforcement for team admins

### Single Sign-On (SSO)
- SAML 2.0 support
- Integration with Okta, Azure AD, OneLogin, Google Workspace
- Just-in-time user provisioning
- SCIM 2.0 for user management

## Compliance & Certifications

- **SOC 2 Type II**: Annual audit by independent third party
- **GDPR Compliant**: EU data protection regulation
- **HIPAA**: Business Associate Agreement available for healthcare
- **ISO 27001**: Information security management (in progress)
- **CCPA**: California Consumer Privacy Act compliant

## Data Centers

- Primary: AWS US-East-1 (Virginia)
- Backup: AWS US-West-2 (Oregon)
- EU Data Residency: AWS EU-West-1 (Ireland) for EU customers
- Uptime SLA: 99.99% for Enterprise plans

## Data Retention & Deletion

### Version History
- Free: No version history
- Starter: 30 days
- Professional: 180 days
- Enterprise: Configurable (up to unlimited)

### Deleted Files
- Trash retention: 30 days (all plans)
- Permanent deletion after 30 days
- Admin can restore files during retention period

### Account Deletion
- Self-service deletion in Settings > Account > Delete Account
- All data permanently deleted within 30 days
- Deletion confirmation sent via email
- Export your data before deletion

## Privacy Practices

### Data We Collect
- Account information (name, email, company)
- Usage analytics (anonymized)
- File metadata (names, sizes, timestamps)
- IP addresses and device information for security

### Data We Don't Access
- File contents (encrypted, we cannot read)
- Passwords (hashed, never stored in plain text)
- Personal files for advertising purposes

### Third-Party Sharing
- We never sell your data
- Third parties limited to essential services:
  - Payment processing (Stripe)
  - Email delivery (SendGrid)
  - Analytics (anonymized, opt-out available)

## Security Incident Response

If we detect a security incident:
1. Immediate investigation and containment
2. Notification within 72 hours (as required by GDPR)
3. Detailed incident report
4. Remediation and prevention measures
